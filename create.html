Create your own images

1.

Develop a concise, abstract description for the desired image, including key elements, themes, or emotions.

2.

Provide your crafted prompt to Midjourney for processing. This can either be done with a discord account through their server, or with an API.

3.

Receive the AI-generated image, review its results, and adjust the prompt if needed for improved outcomes. It typically requires 3-4 attempts with adjusted prompts to achieve the desired image using Midjourney.

Prompt ideas and adjustments

Creating prompts
Prompt Intended Outcome Example Adjustment
Abstract sunset : vibrant colors, warm hues, dynamic sky Abstract image depicting a colorful sunset scene Add “circular shapes” to include geometric elements
Futuristic cityscape : neon light, skyscrapers, night Nighttime image of a futuristic city with neon light Add “hovering vehicles” to enhance the futuristic theme
Forest path : lush greenery, sunlight, peaceful ambiance Serene image of a sunlit path winding through a dense forest Add “woodland creatures” for increased liveliness
